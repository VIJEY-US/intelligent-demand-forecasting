# Intelligent Demand Forecasting System

## Overview
This system predicts product demand to help retailers optimize inventory and reduce stockouts. Leveraging AWS and Python-based machine learning, it offers interactive Tableau dashboards for data insights.

## Features
- **Demand Forecasting**: Machine learning models for accurate predictions.
- **Data Visualization**: Tableau dashboards for trends and insights.
- **AWS Integration**: SageMaker for training and deployment.
